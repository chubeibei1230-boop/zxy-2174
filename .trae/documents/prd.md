## 1. 产品概述

植物插牌内容校对页——为园艺活动准备的纯前端校对工具，帮助园艺团队在活动前高效整理植物名称、养护提示和摆放顺序，确保插牌信息准确无误后交付打印。

- 目标用户：园艺活动策划人员、植物养护负责人
- 核心价值：避免插牌信息错漏，降低活动前校对成本，提升团队协作效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 说明 | 核心权限 |
|------|------|----------|
| 校对员 | 默认角色 | 编辑记录、标记状态、导出打印 |
| 责任人 | 被分配到植物记录的人员 | 查看自己负责的记录、补充养护信息 |

### 2.2 功能模块

1. **校对工作台**：活动信息展示、筛选条、卡片式校对列表、浮动摘要面板、自动检查预警

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 校对工作台 | 活动信息区 | 显示活动名称、日期、地点、参与人数，可编辑保存 |
| 校对工作台 | 筛选条 | 按光照类型（全日照/半日照/耐阴/喜散射光）、责任人、校对状态（待补充/待校对/可打印/暂不展示）、风险等级筛选 |
| 校对工作台 | 卡片式校对列表 | 每张卡片含植物名、拉丁名、适合光照、浇水提醒、展示序号、责任人、校对备注；支持拖拽调序、复制上一条、批量标记 |
| 校对工作台 | 右侧浮动摘要 | 实时统计总记录数、各状态数量、风险项数量，一键生成"打印前校对清单" |
| 校对工作台 | 自动检查面板 | 检测名称重复、序号断档、浇水提醒缺失、同一责任人任务过多，以警示标签形式呈现 |
| 校对工作台 | 导出功能 | 导出 CSV 文件、生成打印版 HTML、打印前校对清单 |

## 3. 核心流程

**主流程**：用户创建活动信息 → 逐条添加植物记录 → 填写/复制植物信息 → 拖拽调整展示序号 → 系统自动检查并提示风险 → 批量标记校对状态 → 生成打印前校对清单 → 导出 CSV 或打印版 HTML

**校对流程**：筛选待校对记录 → 逐条校对信息 → 修改并标记"可打印"或"待补充" → 全部通过后导出

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深森林绿 (#1B4332) + 暖叶绿 (#52B788)，传达自然园艺氛围
- **辅助色**：米白 (#FAF3E0) 作为背景，琥珀色 (#E9C46A) 用于警示，赤陶色 (#E76F51) 用于错误
- **按钮风格**：圆角微凸，主按钮为深绿填充，次要按钮为描边
- **字体**：标题使用 Playfair Display（衬线），正文使用 Noto Sans SC（中文无衬线）
- **布局风格**：左侧卡片列表 + 右侧浮动摘要，顶部信息区横幅，卡片采用微阴影 + 圆角
- **图标风格**：线性图标，使用 lucide-vue-next

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 校对工作台 | 活动信息区 | 顶部横幅，深绿渐变背景，白色文字，可编辑字段以行内编辑形式呈现 |
| 校对工作台 | 筛选条 | 水平排列的下拉选择器 + 搜索输入框，浅色背景带细边框 |
| 校对工作台 | 卡片列表 | 纵向滚动卡片，每张含6个信息字段 + 状态标签 + 操作按钮，拖拽时卡片浮起并加阴影 |
| 校对工作台 | 浮动摘要 | 右侧固定面板，统计数字用大号字体，状态用彩色圆点标识，底部有操作按钮组 |
| 校对工作台 | 自动检查 | 卡片内嵌警示标签（黄/红），摘要面板中汇总警告列表 |
| 校对工作台 | 导出 | 摘要面板底部按钮，点击后下拉菜单选择导出格式 |

### 4.3 响应式设计

- 桌面优先设计，宽屏下左右双栏布局
- 平板端浮动摘要收起为底部抽屉
- 移动端卡片全宽，摘要改为顶部折叠面板

### 4.4 动效设计

- 卡片拖拽：抬起时 scale(1.02) + 阴影增强，放下时弹回动画
- 状态标记：标签出现时 fadeIn + 从左滑入
- 警告提示：黄色/红色标签脉冲闪烁 2 次后稳定
- 摘要面板数字变化：数字滚动过渡效果
